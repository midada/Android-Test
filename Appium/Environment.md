
# 环境

## 1. Python

Python官网：https://www.python.org/

2.X、3.X版本均可

建议python安装的库：

- configParser

- Requests

## 2. Selenium

` pip install selenium `

## 3. Appium

### 3.1 安装Appium

Appium下载地址：https://bitbucket.org/appium/appium.app/downloads/

windows下载exe文件，mac下载dmg文件。

### 3.2 安装Appium Python Client

` pip install Appium-Python-Client `
